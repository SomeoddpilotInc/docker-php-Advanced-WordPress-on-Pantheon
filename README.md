# Docker PHP Pantheon
A PHP Docker image for use with Continuous Integration services deploying to [Pantheon](https://pantheon.io/)

## Includes
* PHP `7.2`
* Composer
* Terminus `2.x`
* rsync
* jq
* wget
* openssl
* ssh
* hirak/prestissimo
* pantheon-systems/terminus-build-tools-plugin
* pantheon-systems/terminus-secrets-plugin
